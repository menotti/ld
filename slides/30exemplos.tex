\frame{{Problema}
    
Um sistema possui dois sensores de temperatura s16 e s18 e dois aquecedores a1 e a2. Seu funcionamento deve ser o seguinte: 
\begin{itemize}
    \item Quando a temperatura estiver abaixo de 16 graus (s16 = 0) os dois aquecedores devem ser ligados.
    \item Quando a temperatura estiver entre 16 e 18 graus (s16 = 1 & s18 = 0) um dos aquecedores deve ser ligado alternadamente a cada entrada nesta faixa de temperatura.
    \item Quando a temperatura estiver acima de 18 graus (s18 = 1) os aquecedores dever ser desligados.
\end{itemize}
}
