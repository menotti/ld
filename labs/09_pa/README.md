# Progressão Aritmética (PA)

O objetivo deste laboratório é implementar um circuito capaz de gerar os termos de uma PA.

## Funcionamento

- `SW[4:0]` é usado para informar a entrada;
- Os termos devem aparecer no *display* em sequência.

## Critérios de avaliação

* **6.0** - Implemente o funcionamento básico na placa.
* **8.0** - Implemente um botão de *reset* sem remover o `por`.
* **10.0** - Implemente um botão de decremento para fazer a **regressão** do mesmo valor.
