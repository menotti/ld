    Laboratório Remoto de FPGAs

    🚨 Isso não é uma simulação!

    O código submetido a este repositório no GitHub é enviado automaticamente ao laboratório remoto e configurado na placa por pelo menos cinco minutos.

    Os pinos de E/S configurados no projeto atual são apenas os seguintes:

    module almost_top(
    input  CLOCK_27, // 27 MHz clock input
    output [5:0] LEDO, // Orange LEDs
    output [7:0] LEDG, // Green LEDs
    output WS2812,     // RGB LED (board)
    output WS2812B,    // REG LED (12 LEDs ring)
    output SERVO);     // PWM signal for servo motor

    Inspecione o código inicial fornecido e tente modificá-lo para fazer alguma iteração sincronizada entre os LEDs verdes e os laranjas. Capture a animação na sua tela com uma gravação no Google Meet e envie ao professor quando solicitado.
